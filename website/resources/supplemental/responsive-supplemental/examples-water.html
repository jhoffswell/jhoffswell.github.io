Percentage of population without access to improved water

This visualization is based on the National Geographic article See Where Access to Clean Water Is Getting Better - and Worse. For our recreations, we reduce the number of data points to limit the impact on the performance of our system and the visualization rendering; that being said, this page can be a little slow, so give it some time to set everything up. This example is particularly exciting because it is interactive! Try using the dropdown below each visualization to highlight a line in the view.

For each visualization, you can use the three dots to (1) save the image, (2) view the Vega-Lite spec, and (3) see the result in the Online Vega Editor.
