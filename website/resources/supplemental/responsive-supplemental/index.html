Example Gallery

This page is a gallery of responsive visualization examples developed in our responsive visualization system. For each example, we show (1) a representative thumbnail of the visualization, (2) the visualization title, (3) a link to the original article or example, and (4) a page of images showing the Vega-Lite spec for each visualization produced in our system. Several examples also include walkthrough video on Youtube for creating the visualizations or the original codes generated during our analysis process.

The thumbnail was created using the responsive visualization system to customize the visualization for display at a smaller size (240x120 pixels). This smaller visualization is also included with the example images. Our responsive visualization system generates a separate Vega-Lite specification for each device-specific visualization. The images page shows each of the visualizations produced through our system. You can play with these examples in the Online Vega Editor using the individual menus on that page.
