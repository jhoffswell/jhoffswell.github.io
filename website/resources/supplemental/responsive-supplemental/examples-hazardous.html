Beijing Air Quality Index (PM2.5)

This visualization is based on data from ChartAccent showing the Beijing Air Quality Index (PM2.5) of December 2014. The provided sample data was used for a Vega-Lite example to show a simple layered bar chart visualization, with an annotation. For this example, we use the Vega-Lite data to (1) reproduce the ChartAccent example with highlighted bands, (2) reproduce the Vega-Lite visualization, and (3) create a thumbnail combining features of both.

For each visualization, you can use the three dots to (1) save the image, (2) view the Vega-Lite spec, and (3) see the result in the Online Vega Editor.

Desktop Visualization (825x585 px)

This example is based on data and a sample visualization from ChartAccent, which shows an annotated line chart visualization of PM2.5 values in December 2014; this recreation shows a smaller sample of data similar to the Vega-Lite visualization.

Phone Visualization - Portrait (332x310 px)

This example is a recreation of one of the Vega-Lite Examples, which uses data from ChartAccent. It is a bar chart of PM2.5 values, with annotated hazardous air levels.

Thumbnail Visualization (240x120 px)

The thumbnail for this visualization combines features of the other two to show a line chart and annotate the threshold for hazardous values.
