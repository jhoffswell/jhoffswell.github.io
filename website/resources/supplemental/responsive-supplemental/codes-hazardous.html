Beijing Air Quality Index (PM2.5)

This visualization is based on data from ChartAccent showing the Beijing Air Quality Index (PM2.5) of December 2014. The provided sample data was used for a Vega-Lite example to show a simple layered bar chart visualization, with an annotation. Note that this example was not included in the original responsive visualization corpus. Therefore, the codes shown on this page correspond only to our example and are not included as part of the counts in the original paper.
